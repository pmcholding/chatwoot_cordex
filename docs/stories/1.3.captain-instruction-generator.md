# Story 1.3: Integração do Captain para Geração de Instruções

## Status
DONE

## Story
**As a** user creating or editing agents,  
**I want** to see a "Criar instruções com IA" button that opens an AI assistant modal (Captain),  
**so that** I can generate high-quality agent instructions through AI-guided conversation instead of writing them manually.

## Acceptance Criteria

1. Add "Criar instruções com IA" button to agent creation/editing form (AddAgent component)
2. Button opens Captain modal with conversational AI interface
3. Captain modal allows multi-turn conversation for instruction refinement
4. Generated instructions are automatically inserted into the agent instructions field
5. Modal provides clear "Use Instructions" and "Cancel" options
6. Captain uses existing OpenAI service infrastructure for generation
7. Modal follows existing Chatwoot design patterns and includes comprehensive i18n
8. Feature works seamlessly in both "create new agent" and "edit existing agent" flows
9. Existing agent creation/editing functionality remains fully intact
10. Captain interaction is logged and tracked for usage analytics
11. Captain instruction generator prompt is configurable through super admin interface at `/super_admin/app_config?config=captain`
