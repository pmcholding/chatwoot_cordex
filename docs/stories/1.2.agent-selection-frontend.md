# Story 1.2: Frontend para Seleção de Modelo de Agente

## Status
Ready for Review

## Story
**As a** user creating new agents,  
**I want** to see a modal offering three creation options: "Usar Modelo", "Criar do Zero", and "Criar com IA",  
**so that** I can choose between predefined templates, manual creation, or AI-assisted agent creation.

## Acceptance Criteria

1. When clicking "Add Agent" button, a selection modal appears with three options using i18n keys
2. "Usar Modelo" option displays available agent templates from the API endpoint with localized text
3. "Criar do Zero" option opens the existing agent creation flow with localized labels
4. "Criar com IA" option opens the Agent Creation form and immediately shows the "Captain" modal for AI-assisted instructions
5. Template selection shows template name, description, and allows user to select one with proper i18n
6. Selected template pre-fills the agent creation form with template instructions
7. AI option flows directly to Captain interaction without requiring manual form filling
8. Modal follows existing Chatwoot design patterns and uses comprehensive i18n for all languages
9. All new text uses i18n keys that follow existing Chatwoot translation patterns
10. Existing agent creation functionality remains fully intact
